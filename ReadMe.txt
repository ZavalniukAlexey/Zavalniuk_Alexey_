Доброго времени суток.

Это последняя версия игры PacMan, которую я успел сделать.

В реализацию добавлены паттерны Наблюдатель и Фабрика*.
Реализовано пользовательское меню.
Реализованы энерджайзеры для пакмена.
Почти полностью реализовано поведение призраков.

*С реализацией фабрики возникла проблема - при добавлении shared_ptr в вектор,
выскакивает ошибка. Она приложена в виде "factory error.png".
В связи с критичностью ошибки и неимение времени на исправление,
решено закомментировать Фабрику с указанием места ошибки.

Исходные файлы находятся в папке "ConsolEngine".
